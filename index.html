Práctica: ajax API request Handlebars.js template

Descripción de la práctica:

Se nos pide realizar una página en la que haciendo uso de jQuery hagamos peticiones AJAX a alguna API pública y mediante plantillas precompiladas de Handlebars.js mostremos la información obtenida dándole formato con Bootstrap 4.

En este caso voy a emplear la api del videojuego MMORPG GuildWars 2: https://wiki.guildwars2.com/wiki/API:Main
